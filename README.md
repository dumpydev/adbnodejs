# ADB installer
Idea from [adb-installer](https://github.com/matejmajny/adb-installer) by matejmajny
# Usage
```
git clone https://github.com/dumpydev/adbnodejs/
```
```
npm i 
```
**or**
```
yarn
```
```
node .
```
# What does it do?
This script will install ADB and add it to path
## Compatibility
- Windows
- MacOS
- Linux
    - ZSH & BASH
    - More coming :)
